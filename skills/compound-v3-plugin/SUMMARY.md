# compound-v3-plugin
A comprehensive Compound V3 (Comet) lending protocol plugin that enables supply, borrow, repay, withdraw operations and COMP rewards claiming across multiple chains.

## Highlights
- Multi-chain support for Ethereum, Base, Arbitrum, and Polygon networks
- Complete lending operations: supply collateral, borrow base assets, repay debt, and withdraw funds
- COMP rewards claiming functionality via CometRewards contract
- Built-in safety features including confirmation gates and dry-run modes
- Automatic debt repayment when supplying base assets
- Real-time market data and position tracking
- Overflow protection for repay operations
- Pre-transaction validation and collateralization checks
