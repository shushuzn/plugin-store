# orca-plugin
Concentrated liquidity AMM on Solana for swapping tokens and querying pools via the Orca Whirlpools CLMM program.

## Highlights
- Swap tokens on Solana through Orca's concentrated liquidity AMM
- Query Whirlpool pools with TVL and fee rate information
- Get real-time swap quotes with price impact estimates
- Built-in security scanning for output tokens
- Automatic price impact protection (warns >2%, blocks >10%)
- Direct integration with Orca's REST API for fast queries
- Support for dry-run mode to preview swaps before execution
- Complete safety checks including slippage tolerance and liquidity warnings
