# pancakeswap-v3-plugin
Swap tokens and manage concentrated liquidity on PancakeSwap V3 across BNB Chain, Base, Ethereum, Arbitrum, and Linea.

## Highlights
- Token swaps via PancakeSwap V3 SmartRouter with best price routing across all fee tiers
- Add concentrated liquidity positions with automatic tick range calculation
- Remove liquidity and collect fees from existing positions
- Real-time price quotes without gas costs
- Multi-chain support: BNB Chain, Base, Ethereum, Arbitrum, and Linea
- View all LP positions for any wallet address
- Pool discovery and liquidity information across fee tiers
- Slippage protection and balance validation for secure transactions
