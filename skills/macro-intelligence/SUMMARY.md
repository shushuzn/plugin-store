# macro-intelligence
Unified macro intelligence feed that aggregates 7+ news sources, classifies macro events, scores sentiment, and generates AI insights via HTTP API.

## Highlights
- Aggregates 7 news sources: NewsNow, Polymarket, Telegram, 6551.io OpenNews, Finnhub, FRED, Fear & Greed Index
- AI-powered event classification with 24+ macro event types (Fed cuts, CPI, whale moves, tariffs)
- Real-time sentiment scoring and regime detection across all sources
- WebSocket and REST data collection with automatic fallback and reconnection
- Live price tickers for SPY, Gold, Silver, BTC, ETH with 24h change tracking
- Cross-source deduplication and sender reputation scoring system
- Dark-theme monitoring dashboard with filtering and real-time updates
- Clean HTTP API for downstream trading skills with flexible signal filtering
