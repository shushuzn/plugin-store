# hyperliquid
A high-performance perpetual derivatives exchange plugin that enables trading, position management, and USDC deposits from Arbitrum to the Hyperliquid L1 blockchain.

## Highlights
- Trade perpetuals with CEX-like speed on Hyperliquid L1
- Place market/limit orders with stop-loss and take-profit brackets
- Check positions, unrealized PnL, and margin usage
- Close positions with one-command market execution
- Set stop-loss and take-profit on existing positions
- Cancel open orders by order ID
- Deposit USDC from Arbitrum via official bridge
- Get real-time market prices for all trading pairs
