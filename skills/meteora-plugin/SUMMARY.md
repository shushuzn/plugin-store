# meteora-plugin
Meteora DLMM plugin for searching pools, getting swap quotes, checking positions, executing swaps, and adding liquidity on Solana.

## Highlights
- Search and filter Meteora DLMM liquidity pools by TVL, APY, volume, and token pairs
- Get real-time swap quotes and execute token swaps via onchainos DEX aggregator
- View detailed LP positions with token amounts computed from on-chain data
- Add liquidity using SpotBalanced strategy with customizable bin ranges
- Remove liquidity with optional position closure to reclaim rent
- Built-in dry-run mode for previewing transactions before execution
- Automatic SOL wrapping and token account creation
- Risk warnings for high price impact and yield pools
