# meteora-plugin
A comprehensive Meteora DLMM plugin for Solana that enables pool searching, swap quotes, position management, token swaps, and liquidity operations.

## Highlights
- Search and filter Meteora DLMM pools by TVL, APY, volume, and token pairs
- Get real-time swap quotes using onchainos DEX aggregator on Solana
- View detailed LP positions with on-chain token amounts computation
- Execute token swaps with slippage protection and price impact warnings
- Add liquidity using SpotBalanced strategy with customizable bin ranges
- Remove liquidity with optional position closure to reclaim rent
- Quickstart wallet analysis with recommended deposit commands
- Support for both native SOL and wrapped tokens with automatic conversion
