# meteora-plugin
A comprehensive Meteora DLMM plugin for Solana that enables pool searching, swap quotes, position management, token swaps, and liquidity operations.

## Highlights
- Search and filter Meteora DLMM liquidity pools by TVL, APY, volume, and fee ratios
- Get real-time swap quotes via onchainos DEX aggregator integration
- View user LP positions with on-chain token amount calculations
- Execute token swaps with dry-run preview and user confirmation
- Add liquidity using SpotBalanced strategy with customizable bin ranges
- Remove liquidity with optional position closing to reclaim rent
- Quickstart command for wallet balance checks and deposit recommendations
- Direct REST API integration with Meteora's DLMM data API for fast pool queries
