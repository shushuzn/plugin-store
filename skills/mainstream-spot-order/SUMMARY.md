# mainstream-spot-order
Multi-chain DEX spot trading system with 6-signal ensemble, auto-research strategy optimization, and per-pair backtesting across SOL, ETH, BTC, BNB, AVAX, DOGE.

## Highlights
- 6-signal consensus trading system (Momentum, EMA, RSI, MACD, Bollinger Bands, BTC overlay)
- Multi-chain support across Solana, Ethereum, BSC, and Avalanche
- AI-driven auto-research for continuous strategy optimization
- Built-in backtesting engine with comprehensive performance metrics
- Paper trading mode for risk-free validation
- Real-time data collection and dashboard at localhost:3250
- TEE-signed transactions via OKX Agentic Wallet (no local private keys)
- Zero pip dependencies - pure Python stdlib implementation
