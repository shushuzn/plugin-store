# aave-v3-plugin
Lend and borrow crypto assets on Aave V3 — the leading decentralized liquidity protocol.

## Highlights
- Supply assets to earn yield on Ethereum, Polygon, Arbitrum, and Base
- Borrow against collateral with real-time health factor monitoring
- Automatic WETH wrapping when supplying ETH-based assets
- Smart approval handling for seamless ERC-20 transactions
- View live interest rates and APYs across all supported markets
- Monitor positions with detailed supply/borrow breakdowns
- Enable/disable collateral and set E-Mode for optimized borrowing
- Claim protocol rewards with integrated position management
