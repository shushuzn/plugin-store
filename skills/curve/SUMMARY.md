# curve
Curve DEX plugin for swapping stablecoins and managing liquidity across Ethereum, Arbitrum, Base, Polygon, and BSC.

## Highlights
- Swap stablecoins on Curve Finance across 5 major chains
- Add and remove liquidity from Curve pools
- Query pool information, APY rates, and TVL data
- Get LP token balances and positions
- Preview transactions with dry-run mode before execution
- Automatic slippage protection and price impact warnings
- Support for both proportional and single-coin withdrawals
- Integration with onchainos wallet for secure transaction execution
