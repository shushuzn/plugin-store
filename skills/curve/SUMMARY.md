# curve
Curve DEX plugin for swapping stablecoins and managing liquidity across Ethereum, Arbitrum, Base, Polygon, and BSC.

## Highlights
- Swap stablecoins with low slippage on Curve pools
- Add and remove liquidity from Curve pools
- Query pool information, APYs, and TVL data
- Check LP token balances across supported chains
- Get swap quotes with customizable slippage tolerance
- Support for 5 major chains (Ethereum, Arbitrum, Base, Polygon, BSC)
- Automatic pool selection by TVL for optimal liquidity
- Dry-run preview for all write operations before execution
