# compound-v3
A Compound V3 (Comet) lending plugin that enables supplying collateral, borrowing/repaying the base asset, and claiming COMP rewards across multiple chains.

## Highlights
- Multi-chain support (Ethereum, Base, Arbitrum, Polygon)
- Supply collateral assets to earn interest
- Borrow base assets against collateral
- Automatic debt repayment when supplying base assets
- Claim COMP token rewards
- Dry-run mode for previewing transactions
- Built-in safety checks and overflow protection
- Integrated with onchainos wallet for secure transaction execution
