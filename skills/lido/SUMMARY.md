# lido
Stake ETH with Lido liquid staking protocol — stake, manage withdrawals, and track staking rewards on Ethereum mainnet.

## Highlights
- Stake ETH to receive stETH (liquid staking token)
- Request withdrawals from stETH back to ETH
- Track staking APY and rewards in real-time
- Manage pending and finalized withdrawal requests
- Claim ETH from completed withdrawals
- Check stETH balances with rebasing calculations
- Support for referral addresses when staking
- All operations require user confirmation for security
