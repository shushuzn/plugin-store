# pancakeswap-clmm-plugin
A PancakeSwap V3 CLMM farming plugin that enables staking LP NFTs into MasterChefV3, harvesting CAKE rewards, and collecting swap fees across BSC, Ethereum, Base, and Arbitrum.

## Highlights
- Stake V3 LP NFTs into MasterChefV3 to earn CAKE rewards
- Harvest CAKE rewards without withdrawing staked positions
- Collect swap fees from unstaked LP positions
- View pending rewards and active farming pools
- Support for BSC, Ethereum, Base, and Arbitrum networks
- Preview all operations before execution with dry-run mode
- Automatic wallet integration via onchainos CLI
- Complementary to pancakeswap-v3 plugin for complete V3 workflow
